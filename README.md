# 桥架弯头计算器 (Bridge Calculator)

> 离线桥架弯头计算工具 · 68种做法 · 3D实时渲染 · 完全离线

陈工桥架计算器 **Android 原生离线版** —— 包含全部 68 种弯头做法、万能计算公式、3D 实时模型展示。

- 📐 **68 种弯头做法** 全覆盖（爬坡/水平/组合翻弯/变径/三通/四通/任意角）

- 📱 **Android 原生**（Kotlin + Jetpack Compose）
- 🚀 **GitHub Actions 自动构建** + Release 发布

## 📐 68 种弯头分类

1. **上下爬坡** (6种)：左/右爬坡、90度成品、90度垂直、左右水平爬坡
2. **水平转向** (3种)：左/右水平、90度水平角
3. **组合翻弯** (10种)：45度+45度组合、外/内翻、30度组合
4. **特殊造型** (10种)：S型、过桥、抱梁、漏斗、堵头等
5. **变径** (8种)：90度变径、漏斗变径、边高变径、异径
6. **三通/四通** (10种)：水平/垂直/斜/异径三通、四通
7. **折角+盖板** (5种)：90度折角、一体弯、各种盖板
8. **任意角** (2种+万能)：任意角度弯头

## 🧮 计算公式

### 万能公式
```
X = b × tan(α/2)
```
- `X` = 下料口尺寸
- `b` = 桥架帮高（边高）
- `α` = 桥架与平面爬坡的夹角

### 速算系数

| 角度 | 切口中线距离 | 斜边 |
|------|-------------|------|
| 30°  | × 0.5       | × 2  |
| 45°  | × 0.8       | × 1.4 |
| 60°  | × 1.0       | × 1.15 |
| 90°  | 切两个45°合成 | -    |
